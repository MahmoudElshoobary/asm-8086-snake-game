# 🐍 Snake Game in 8086 Assembly

A classic Snake game built entirely in 8086 Assembly Language using direct video memory manipulation and BIOS interrupts.

---

## 🎮 Features

- Real-time snake movement
- Continuous directional movement
- Snake growth after eating apples
- Random apple spawning
- Screen wrapping mechanics
- Optimized rendering system
- BIOS timer-based game loop
- Direct VGA text mode rendering

---

## ⚙️ Technologies Used

- 8086 Assembly Language
- EMU8086
- BIOS Interrupts
- VGA Text Mode Memory (`0B800h`)

---

## 🧠 Concepts Used

This project demonstrates:

- Direct video memory programming
- Keyboard interrupt handling
- BIOS timer interrupts
- Real-time game loops
- Collision detection
- Dynamic arrays in assembly
- Efficient rendering techniques
- Game state management
